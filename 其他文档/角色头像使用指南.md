# 文件命名与链接使用指南

本文档旨在说明本项目中角色立绘文件的命名规则，以及如何通过链接访问它们。

## 链接结构

所有图片都已上传至您的 Cloudflare R2 存储桶，并可通过您的自定义域名访问。链接的基本结构如下：

`https://onmm.top/image/char/{角色ID}/{表情ID}.png`

**组成部分:**
- **基础 URL**: `https://onmm.top/image/char/` (这是固定的)
- **{角色ID}**: 代表角色的唯一英文ID。
- **{表情ID}**: 代表该角色某个表情的唯一英文ID。

## 使用示例

1.  **获取博丽灵梦的微笑表情:**
    - 角色ID: `reimu`
    - 表情ID: `smile`
    - 最终链接: `https://onmm.top/image/char/reimu/smile.png`

2.  **获取魂魄妖梦2号立绘的说话表情:**
    - 角色ID: `youmu_2`
    - 表情ID: `talk`
    - 最终链接: `https://onmm.top/image/char/youmu_2/talk.png`

3.  **获取秦心的愤怒说话表情 (来自般若面具):**
    - 角色ID: `kokoro`
    - 表情ID: `angry_talk`
    - 最终链接: `https://onmm.top/image/char/kokoro/angry_talk.png`

您可以根据以上规则和列表，自由组合来获取任何您需要的图片。
